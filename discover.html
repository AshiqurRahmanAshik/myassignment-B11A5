Question-1: What are the different ways to select an element in the DOM?

getElementById() , getElementsByClassName() and getElementsByTagName() select elements based on ID , class or tag name, respectively. querySelector() and querySelectorAll() are the most flexible options since they allow the use of CSS selectors.

Question-2:What is the difference between innerHTML,innerText and textContent?

innerHTML is used when working with HTML content (tags) inside an element. innerText is used when you need the visible text content, respecting CSS styles. textContent is used when you need the raw text content of an element, excluding tags and ignoring CSS styles (faster than innerText).

Question-3:What is event delegation in DOM?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element rather than adding individual event listeners to multiple child elements. The parent element handles events for its child elements. This approach is efficient and takes advantage of event bubbling to manage events on dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in JavaScript that describes how events propagate through the DOM (Document Object Model) tree. When an event is triggered on an element, it doesn't just affect that element. Instead, the event "bubbles up" from the target element to its parent elements, eventually reaching the root of the document. This is called event bubbling.
